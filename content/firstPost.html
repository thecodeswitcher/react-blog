Top 3 Transferable Skills to Learn During the Crisis

The Most Adaptable Thrive

“The green reed which bends in the wind is stronger than the mighty oak which breaks in a storm.” -Confucious

The corona virus epidemic has caused global disruption. Why is this happening on such a large scale economically? People have had to adapt quickly.

Logistically, in-person meetings are no longer possible. In fact, any in-person work has been significantly reduced. Logically, these are the most negatively affected people.

If you're now working from home and find yourself with a much more flexible, commute-free schedule, here are some profitable skills you can learn to come out ahead during the crisis (and no they're not all coding).

Data Analysis

Data Science and Machine Learning have become almost cliché buzzwords that all employers want but few understand.

Data analysis is a necessary pre-requisite that doesn't require complex math, but has large implications.

Data analysis features prominently in the news, vlogs, articles, and any kind of informational content. The ability to make an argument supported by numbers has been rapidly growing in demand. Researching trends are even more important especially when the stock market and wider economy are in a state of flux and uncertainty.

What tools do data analysts use?

  • Excel
  • SQL
  • Python/R
Excel

I bet you weren't expecting this article to promote learning spreadsheets, but even the most seasoned data scientist will occassionally have to work with Excel (however lightly) for one project or another.

How deeply you have to learn Excel depends on the company and the project, but there are basics that everyone should know (even if you're not a data analyst).

  • Charts (bar, line, scatter plots)
  • Filtering
  • Pivot tables
  • Formulas (SUMIF, AVGIF, VLOOKUP)

This isn't an exhaustive list, but the main idea is to master these 3 competencies:

  • Data cleaning: get messy data into the form you need
  • Insights derivation: If you have 10,000 rows of data, what trends are we seeing
  • Data presentation: Be able to present these insights visually (and verbally), so non-technical people can understand them.

We could probably stop the list here because every single industry needs people that can clean, reshape, and present data. If you master these in Excel, it's all just learning a new tool.

SQL

After getting comfortable with the 3 basic data competencies, SQL is a great next step to be able to pull large amounts of data.

Develop some of the most frequently used skills:

  • JOINs: Combine one dataset to another.
  • Filtering: Just like in Excel, learn how to get specific sub-populations and the data you need.
  • Aggregate functions and grouping: Like in Excel, you can get averages and sums from an entire dataset. But now you can get averages by a particular group. (e.g. Average time on Reddit by age group)

If you search on Indeed or LinkedIn with just the keyword SQL, you'll get a good idea of the skill's demand and transferability.

Python/R

I want to preface this section by saying that with Excel and SQL alone, you already have put yourself in a very strong position career-wise. Those two skills can set you up to be a business analyst that's constantly getting LinkedIn DMs to interview at other companies.

Still, learning Python or R will help you automate a lot of repetitive tasks.

I have a friend that works at a trading firm as a data analyst. Python/R is not a pre-requisite, but every day, the analysts have to give a report on certain stocks and indicies.

Other analysts just use Excel and manually produce the report every day. My friend has set a process to run on his computer every day before work that pulls the data, produces the chart, and emails it out.

People prefer reading his reports because they have minimal errors and come at a consistent time.

Even people that have been making spreadsheets for 10+ years make mistakes. If you learn the basics of Python or R, you can write a script to produce a report once and have that run everyday without mistakes.

As a minimum, learn these skills:
  • Reading .csvs and .xlsx documents
  • Cleaning data
  • Producing charts
  • Daily running of scripts (i.e. cron jobs)

Coding

I'm going to be that guy now. No, this section isn't going to be #learntocode.

Instead, we're going to look at coding from a different angle: creativity and building.

For me, coding and building projects feel like playing with Legos. Once you learn how to fit the pieces together, you can make any idea you have into a reality.

When I learn a new technology, I start getting ideas about how I could make my life easier with it. For example, when we learned to build basic web applications with machine learning back-ends, I started coming up with ideas like

  • automating political research
  • sending daily reports on how the stock market reacts to the news
  • making Spotify mixtapes for a crush

Before talking about the core competencies, here are some resources to learn programming basics (not afiliate links, I used these myself when I started learning):

  • Codecademy: Gamified learning with a free version
  • Udemy: frequent sales on great courses
  • Team Tree House: Monthly subscription for a wide breadth of skills

I suggest you try Codecademy first to see if you even like programming. It would be great to have a basic knowledge to supplement any other skills you already have, but if you're thinking about a career change, make sure that you actually enjoy what you'll be doing for 2000 hours a year.

The core competencies that I suggest you learn will surprise you because they're not basic programming skills:

  • Meta learning: learn how you learn best.
  • How to Google: Seriously, learning how to use Google when you find a bug in your program will save you a lot of time. It's really how to research.
  • Problem solving: If someone needs a new tool, you can buckle down, do the research, learn new skills (hmmm...aren't those the other two skills?), and build a solution.

Programming is really about those three competencies. Yes, you do have to learn how to declare variables, use loops, and work with files. But adaptable, in-demand workers in any field have mastered those 3 competencies.

What does that mean for you?

To be fair, the pandemic has affected everyone.

But there are some that aren't struggling as much as the rest of the workforce.

Most office jobs have allowed people to work from home. A friend actually commented that "Now we'll really see which meetings could have been emails." Those lucky to be in those positions can indeed continue to work, but, if the business' main source of revenue relies on in-person experiences, layoffs could happen at any time.

Transferable skills are some of the most valuable commodities in a crisis (and arguably even beyond).

Here's why:
  1. They are portable.
  2. They make you independent.
  3. You can build with them.

Soft skills and networking are absolutely essential as well, but you can learn both at the same time.

In fact, you should. If you don't pay attention to soft skills, you may get passed up for promotions or you might not be able to advertise yourself as effectively if you're self-employed.

On the flip side, if you're only focused on networking and people skills but have no transferable skills, crises and layoffs will hit you hard.

Those that have both can frame any career transitions as opportunities to skill up, meet new people, and come out even more employable on the other end of this.
